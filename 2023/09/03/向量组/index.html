\left \{ \alpha_1, ..., \alpha_n \right \} 线性无关, 表出组 (\beta_1, ..., \beta_m) = (\alpha_1, ..., \alpha_n) (\vec{k}_1, ..., \vec{k}_m) \\ k_* 是 \left \{ \alpha_1, ..., \alpha_n \right \} 线性组合系数, 证明 \left \{ \beta_1, ..., \beta_m \right \} 无关 \\ 可以将 \left \{ \beta_1, ..., \beta_m \right \} 线性组合表示回 \left \{ \alpha_1, ..., \alpha_n \right \} 的线性组合, 利用其系数都为 0 \\ 也可以判断 B = A P 的秩, \therefore \left \{ \beta_1, ..., \beta_m \right \} 无关 \iff r(AP) = m \\ \left\{\begin{matrix} \because A 列满秩 \therefore r(AP) = r(P) \\ r(B) \le r(A) \and r(B) \le r(P) \end{matrix}\right.
